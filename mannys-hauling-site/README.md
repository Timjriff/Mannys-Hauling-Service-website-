# Manny's Hauling Services Website

Premium Next.js 15 website for Manny's Hauling Services, a junk removal company in Santa Ana, CA.

## Tech

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons
- SEO metadata, Open Graph, sitemap, robots, LocalBusiness JSON-LD, FAQ JSON-LD

## Run locally

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:3000
```
